XmlElement

sealed interface XmlElement

The interface used to represent the XML elements used in a document (such as tags and their content). This implements 2 design patterns simultaneously: the Composite design pattern (where the XmlTag is the Composite, and the XmlTagContent is a Leaf), and the Visitor design pattern.
