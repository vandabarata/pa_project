XmlTagWithContent

data class XmlTagWithContent(var name: String, val parent: XmlTag, val content: String, var tagAttributes: MutableMap<String, String> = mutableMapOf()) : XmlElement

This class represents an XML Tag with textual content. The name of the tag represents what shows as the and the content is what shows inbetween. It's considered the leaf element, as it can never have other children elements.
