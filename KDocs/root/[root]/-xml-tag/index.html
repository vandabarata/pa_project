XmlTag

data class XmlTag(var name: String, val parent: XmlTag? = null, var tagAttributes: MutableMap<String, String> = mutableMapOf()) : XmlElement

Oject that implements the XmlElement interface, the XmlTag is considered the Composite object, being considered the element that nests others and can have them as children. This is where the visits start.

An XmlTag must be given a name, and can be given another XmlTag as a parent. It may also include attributes, which are values mapped to certain keys that might show up as on an XML Document.
